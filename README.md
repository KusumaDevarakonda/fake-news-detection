# Fake News Detection Using Machine Learning

This project uses Machine Learning and Natural Language Processing (NLP) techniques to classify news articles as Fake News or Real News.

## Features
- Text preprocessing and cleaning
- TF-IDF vectorization
- Logistic Regression model
- Fake vs Real news prediction
- Accuracy evaluation

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- NLP

## Dataset
Fake and Real News Dataset from Kaggle.

## Project Workflow
1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis
4. Text Preprocessing
5. TF-IDF Vectorization
6. Model Training
7. Prediction and Accuracy Evaluation
